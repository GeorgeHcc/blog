# 第二章 进程的描述与控制
::: details
并发与并行的区别：

- 并发是指<mark>单个处理器</mark>同时处理多个不同任务，并行是指<mark>多个处理器/多核处理器</mark>同时处理多个不同的任务
- 并发是逻辑上的同时发生，从微观上看，由于指令的执行速度非常快，以至于给人的感觉像是同时发生；而并行则是物理上的同时发生，无论是在微观下还是宏观下都是同时执行的
:::
